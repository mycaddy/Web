• マイキャディWT V8の更新手順

  1. 위의 Select Model 옵션에서 업그래이드 하실 단말기의 모델과 언어를 선택하세요.

  2. 를 클릭하세요.

  3. 다운로드가 완료되면 하단에 아래와 같이 "Completed"로 표시됩니다.

  4. 단말기가 정상적으로 연결되어 있고 아래와 같이 "Autoupgrade" 옵션이 체크되어있으면 자동으로 업그레이드 과정이 진행됩니다.

  5. 단말기와 USB케이블이 정상적으로 연결되었다면.
    아래 채크박스에 연결 표시가 됩니다.

  6. 다운로드가 완료되면 버튼이 활성화 됩니다.
    (위 Auto upgrade 옵션이 체크되어 있지 않고 다운로드 이후 제품을 연결 한 경우 수동으로 업데이트가 가능합니다.)

  7. 업데이트가 완료되면 하단에 아래와 같이 "Completed"로 표시됩니다.

  8. 모두 완료되면 제품과 케이블을 분리하시고,실내가 아닌 실외에서 "GPS"를 수신하면 현재 시간이 세팅됩니다.
